## Website Style Principles
| Theme                  | Description                                                              | Application                         |
| ---------------------- | ------------------------------------------------------------------------ | ----------------------------------- |
| **Symmetry & Balance** | Every layout mirrors geometric clarity.                                  | Grids, iconography, spacing rhythm. |
| **Refined Light**      | Use gradients and glows sparingly to imply intelligence, not excess.     | Hero backgrounds, hover states.     |
| **Organic Geometry**   | Petal shapes recur subtly in buttons, section dividers, and transitions. | SVG motifs and animations.          |
| **Technical Grace**    | Combine elegance with function - performance meets design.               | Code samples, runtime previews.     |
