## Typography
| Type               | Font                  | Usage                                     | Weight           |
| ------------------ | --------------------- | ----------------------------------------- | ---------------- |
| **Display / Logo** | `Aeonik` or `Satoshi` | Headings, hero text, wordmark “oro”       | Bold / SemiBold  |
| **Body**           | `Inter`               | Paragraphs, documentation, UI copy        | Regular / Medium |
| **Mono (Code)**    | `JetBrains Mono`      | Code blocks, CLI, developer documentation | Regular          |

## Ratios
- `h1`: `48px / 1.1 line-height`
- `h2`: `32px / 1.2 line-height`
- `h3`: `24px / 1.3 line-height`
- `body`: `18px / 1.6 line-height`
- `caption`: `14px / 1.6 line-height`
